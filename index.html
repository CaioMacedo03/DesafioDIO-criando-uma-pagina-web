Inspetor de Elementos

O Inspetor de Elementos é uma ferramenta essencial para programadores, especialmente aqueles que trabalham com desenvolvimento web.Pois permite identificar problemas no HTML, CSS e JavaScript que podem ser corrigidos no próprio navegador e ajuda a localizar alguns erros de layout ou estilo.


Estrutura Básica do HTML

A estrutura básica de um documento HTML é como o esqueleto de uma página web. É simples, mas cada parte tem sua função específica para ajudar o navegador a entender o que deve ser exibido e como.

Esta estrutura é formada principalmente pelas Tags head, title, body.


Falando sobre Tags

Tags são como pequenos rótulos usados para organizar e estruturar o conteúdo em páginas da web. Elas aparecem entre os símbolos <> e servem para indicar o que cada parte da página representa ou como ela deve funcionar. Por exemplo, uma tag pode dizer que um pedaço de texto é um título, que outro é um parágrafo, ou até que uma imagem precisa ser exibida.


Atributos das Tags

Os atributos das tags são como detalhes extras que adicionamos para personalizar ou ajustar o comportamento e a aparência de um elemento em uma página web. Eles ficam dentro da tag e geralmente funcionam como pares de "nome" e "valor", assim: nome="valor". É como se fossem instruções adicionais para deixar cada elemento mais funcional ou único.


Textos

Tags de texto no HTML ajudam a estruturar e estilizar o conteúdo textual, garantindo que ele seja exibido de forma organizada e acessível, além de facilitar a leitura e a navegação.Como por exemplo a tag h1(titulo) ou a tag h2 (subtitulo).


Listas ordenadas e não ordenadas

Listas ordenadas e não ordenadas são tipos de listas em HTML que ajudam a organizar informações em itens distintos. A principal diferença entre elas é o modo como os itens são apresentados:

A lista ordenada é uma lista em que os itens são numerados ou exibidos em ordem específica (números, letras ou algarismos romanos, dependendo do estilo).

A lista não ordenada é uma lista em que os itens são exibidos com marcadores (como pontos, círculos ou quadrados) em vez de números.


Links

No HTML, os links são criados com a tag , que significa "âncora". Eles servem para conectar páginas da web, direcionar para arquivos ou imagens, levar o usuário a uma parte específica de uma página, ou até realizar ações, como abrir um programa de e-mail. Os links funcionam como "portas" que os usuários podem clicar para navegar entre conteúdos ou acessar informações de forma simples.
